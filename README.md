# The solx fork of the LLVM Compiler Infrastructure

Welcome to the **solx** project!

This repository contains a fork of LLVM used by the **solx** compiler.

To know more about the project, please visit the official **solx** repository and documentation:

- [solx Compiler Repository](https://github.com/NomicFoundation/solx)
- [solx Documentation](https://nomicfoundation.github.io/solx/latest)
- [Building from Source](https://nomicfoundation.github.io/solx/latest/#building-from-source)

If you are looking for the original LLVM repo, please follow this link to the upstream:

- [Upstream LLVM Repository](https://github.com/llvm/llvm-project)
